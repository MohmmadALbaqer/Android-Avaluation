### Installation

```
$ git clone https://github.com/MohmmadALbaqer/Android-Avaluation.git
$ cd Android-Avaluation
$ chmod +x *
$ pip3 install -r requirements.txt
$ sudo bash install.sh
$ sudo python3 Android.py
```
### To monitor the Android system
```
$ bash scan.sh
```
